# Overlay Render — EP-133 K.O. II visualization, vol. 1

A tool for rendering video with overlaid text, sequencer visualization, and animation.

Code was vibecoded in a hurry over 5 evenings with the help of DeepSeek.  
Data-driven configuration is tailored for the track "From the Ruins".

---

### Installation & Usage

pip install -r requirements.txt  
Place your source video in the video/input folder.  
In config.py, set RENDER_MODE:

* False — preview mode
* True — render mode

On first run, the script will download and install portable ffmpeg.
